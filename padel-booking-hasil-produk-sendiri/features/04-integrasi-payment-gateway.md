# Integrasi Payment Gateway

**Fase:** Fase 2 | **Prioritas:** medium

## Deskripsi
Otomatisasi verifikasi pembayaran via Midtrans/Xendit.

## Tujuan
Menghilangkan proses cek manual oleh admin.

## Selesai Bila
- User diarahkan ke halaman payment gateway
- Webhook menerima status pembayaran
- Status booking otomatis terupdate

## Sub-fitur
### Payment Link
Generate link pembayaran otomatis.

### Webhook Listener
Menerima update status dari gateway.
