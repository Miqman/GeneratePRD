# Katalog & Grid Jadwal Real-time

**Fase:** Fase 1 | **Prioritas:** high

## Deskripsi
Tampilan daftar venue dan grid ketersediaan slot waktu per jam.

## Tujuan
Pemain melihat jadwal kosong real-time tanpa login.

## Selesai Bila
- Daftar venue dan detail lapangan tampil
- Grid jadwal akurat sesuai database
- Waktu muat < 1.5 detik

## Sub-fitur
### Daftar Venue
Menampilkan venue dan detail lapangan.

### Grid Jadwal
Status ketersediaan slot waktu per jam.
