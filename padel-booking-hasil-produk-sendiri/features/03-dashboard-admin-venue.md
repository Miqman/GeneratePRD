# Dashboard Admin Venue

**Fase:** Fase 1 | **Prioritas:** high

## Deskripsi
Panel verifikasi untuk menyetujui atau menolak booking.

## Tujuan
Admin memverifikasi bukti transfer dan kelola status jadwal.

## Selesai Bila
- Daftar booking pending tampil
- Admin dapat melihat foto bukti transfer
- Status booking dan slot terupdate saat disetujui

## Sub-fitur
### Daftar Booking
Tabel reservasi menunggu verifikasi.

### Verifikasi Pembayaran
Aksi setuju/tolak berdasarkan bukti transfer.
